''' python
函数值的传参
1，位置传参
2，关键数传参
3,*序列传参
4，**key传参

2。。。指的是传参是，按着形参的名称来赋值
实参形参按名称匹配'''
def s3(a,b,c):
    pass
    print(a)
    print(b)
    print(c)
s3(a=2,c=3,b=5)
#这样的赋值是可以换位置的
但是不能缺少，和多出未定义的关键字
3.。。。序列传参
序列类型（list tuple str）来传递，但是序列的个数与参数个数一致
比如：
lst1=[1,2,3]
s3(*lst1)
元组和字符串同理
4，，，字典关键字传参
dic1={'a':33,'b':22,'c'=44}
s3(**d1)
和按名称匹配是类似的，名字要一一对应，健名是字符串

1函数的缺省参数
当函数的形参被赋予默认值，那么调用时如果没有实参传递，就用默认值
如果一个参数缺省，右侧全部缺省
2不定长参数
把一个元组字典，当作参数，对相应的元组或字典进行操作
元组是单*号
字典树=是**
如
def myfun(**args)
    print(len（args）)
    for k,v in args.items()：
    print(k,v)
*号相当于一个特化符，出现*后参数都要关键字赋值，或元组
myfun('age'=18)
赋值=，变成字典后是：
其实就是把多个关键字赋值，合成一个字典，与元组类似
